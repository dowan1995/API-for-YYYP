# 悠悠有品-饰品市场获取固定商品的最近租赁成交记录

获取悠悠有品平台下固定goods_id的最近租赁成交记录，并追加写入Excel表格。

以程序运行时间戳作为唯一值，Excel表格中可以使用筛选区分。
